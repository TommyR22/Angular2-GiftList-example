# Gift List

Wedding gift list service made with Angular2, nodejs.

## Getting Started

front-end:
```
git clone https://github.com/TommyR22/Angular2-GiftList-example.git
cd ..
npm install
ng serve
```

backend:
```
node server.js
```

# Testing

```
http://localhost:4200
```
